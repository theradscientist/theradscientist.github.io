CO2-Toluene Core Cleaner Program

A CO2-Toluene core cleaner is designed to clean rock samples of their organic matter under high temperature and pressure. This is done so that an accurate pore volume can be determined. I designed a LabVIEW program that uses Queues to control a core cleaner for long periods of time.

The program is able to read pressures from transducers, read temperatures, and read tank levels in order to control 7 air actuated valves in the system. The program automatically raises CO2 pressure, fills tanks with toluene, holds for a defined period of time at temperature, and safely releases pressure.
