Stress Dependent Program

The 3D stress dependent instrument is designed to apply pressure on a rock sample from 10 different directions in order to best simulate in-situ stress conditions. Once under stress, acoustic velocities are measured through each stress direction to determine the anisotropy of the rock. I designed this LabVIEW program to automatically acquire all 30 acoustic signals and provide tools to help with analysis.

The program sends commands to a Keithley switchbox to switch between each pair of piezeo-electric crystals, inducing 3 waveforms for each direction. The program then retrieves the waveforms from a Keysight oscilloscope and displays the data for the user to analyze. Once arrival times have been selected by the user, the program generates and saves an Excel spreadsheet with the compiled data. For more information, you can view the user manual I created here.
